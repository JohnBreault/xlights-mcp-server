# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public issue.**

Instead, please use [GitHub's private vulnerability reporting](https://github.com/JohnBreault/xlights-mcp-server/security/advisories/new) to submit your report. This ensures the issue can be triaged and addressed before public disclosure.

You can expect:

- An acknowledgment within **48 hours**
- A plan for resolution or mitigation within **7 days**
- A public disclosure after a fix is available

Thank you for helping keep this project safe.
